How to find available parking garage


1. Select the desired parking permit from the Home Page.

2. You are redirected to a map view with every parking garage
and parking lot relevant to your permit.

3. You can visually identify how busy a parking garage or parking
lot is based on the color gradient of the circle located at each zone.
Ex.




4. Once you determine which parking garage is more convenient,
you can view the garage availability per floor by clicking
on the “Availability” link for each parking garage.


How to find A Class

1. Select the Find My Class button in the Home Page.

2. Select the relevant class from the dropdown and press Find button.

3. You are redirected to a map view with a locator for the class selected.
